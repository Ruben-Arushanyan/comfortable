The **$Value** is the main class that provides the common methods to work with JavaScript value regardless of the value type․

## Import

```
import {$Value} from 'comfortable/$Value'
```

## Simple Usage Example

```js
import {$Value} from 'comfortable/$Value'

$Value('Hello').isString() // true
```
