The **$value()** constructor is used to create [$value](./index.md) objects.

## Examples

```js
import {$value} from 'comfortable/$value'

$value('Hello')

$value(123)

$value(true)

$value(null)

$value({a: 1})

$value(() => {})
```

## Syntax

### `$value(value)`

#### Arguments

- **value** `<any>`  
    Any JavaScript value.
